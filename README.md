## 📋 Overview

Omni Calculator is a two-in-one web tool that combines a scientific calculator with a unit converter. The goal was to build a clean, functional, and responsive utility that covers everyday math needs — from basic arithmetic to trigonometric functions — alongside a versatile converter supporting 7 measurement categories.

---

## ✨ Features

### Calculator
- Basic arithmetic: addition, subtraction, multiplication, and division
- Exponentiation (`^`) and nth-root (`√`)
- Scientific functions: `sin`, `cos`, `tan`, `log`, `ln`, `|x|`
- Full expression display with real-time editing
- Parentheses support for grouped expressions
- Negative number input
- Backspace support for correcting entries
- **Keyboard support** — operate the calculator without touching the mouse
- **Calculation history** — review and clear past operations via a modal

### Unit Converter
- 7 conversion categories:
  - 📏 Length
  - ⚖️ Mass
  - 🌡️ Temperature
  - ⬛ Area
  - 🧊 Volume
  - 💨 Speed
  - ⏱️ Time
- Swap button to instantly reverse the conversion direction
- Handles temperature conversions (Celsius, Fahrenheit, Kelvin) with dedicated logic

## 🚀 Getting Started

No installation or build step required. Just clone the repository and open the project in your browser.

```bash
git clone https://github.com/RicardoLF1914/omni-calculator.git
cd omni-calculator
```

Then open `index.html` in any modern browser.

> **Tip:** For the best experience with linked assets, serve the project through a local server (e.g., the VS Code Live Server extension) rather than opening the file directly.
